Este exemplo usa media queries para reorganizar as imagens em diferentes tamanhos de tela: para telas maiores que 700px de largura, ele mostrará quatro imagens lado a lado; para telas menores que 700px, mostrará duas imagens lado a lado. Para telas menores que 500px, as imagens serão empilhadas verticalmente (100%).
